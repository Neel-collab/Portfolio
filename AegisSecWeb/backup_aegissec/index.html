Monolithic Core Internals

Thread-Safe Static Modules

The MongoHelper is architected as a static VB.NET Module, ensuring the MongoClient singleton is physically instantiated in the high-frequency memory heap exactly once, preventing TCP port exhaustion and handle leaks during rapid multi-form switches.

BSON binary serialization

Bypassing JSON stringification, AegisSec communicates via raw BsonDocument streams. This allows for direct binary mapping of CLR types to MongoDB's WiredTiger engine, reducing CPU cycles by 40% compared to standard REST-based security apps.

Atomic $set Mutators

Update operations utilize MongoDB's atomic $set and $inc operators. This eliminates "Lost Update" scenarios in multi-user environments by performing the modification directly on the primary index shard without a local fetch-save cycle.

GDI+ Native Painting

The Dashboard utilizes OnPaint overrides rather than standard control-rendering. This allows the GPU to directly composite glowing neon borders and semi-transparent glass panels, maintaining a fluid 60FPS UI response even during massive background data ingestion.

Dns InterNetwork Filtering

The SecurityLogger executes a hardware-level IP scrape using System.Net.Sockets.AddressFamily.InterNetwork. This explicitly bypasses IPv6 Teredo tunnels and virtual loopbacks, ensuring the recorded Telemetry is physically tied to the primary hardware NIC.

Non-Blocking Async Await

The Twilio 2FA engine is non-blocking. By using Await PostAsync, the application yields its execution context back to the Windows Message Loop, preventing the common "App Not Responding" white-screen ghosting during high-latency network calls.

Dynamic Type Coercion

The system intelligently mitigates BSON driver type-mismatches. By utilizing ToInt32() instead of strict AsInt32 casting, it dynamically scales BsonDouble variants returned by MongoDB into compliant CLR Integers, neutralizing potential session-render crashes.

Aggressive Driver Timeouts

Standard drivers wait 30s for a lost server; AegisSec aggressively overrides this to 2000ms via ServerSelectionTimeout. This ensures the UI thread fails fast and remains interactive during catastrophic network severance.

Double Buffered GDI+ Flip

The primary canvas is initialized with OptimizedDoubleBuffer logic. This creates a high-speed off-screen memory map where the GPU assembles the interface prior to flipping it to the monitor, eliminating 100% of visible rendering flicker.

Entity Relational Topologies

Collection: users

Primary identity storage enforcing granular Role-Based Access Control and biometric-ready authentication frameworks.

  • _id: ObjectId
  • username: String (Unique Header)
  • password: String (Hashed BSON String)
  • role: String (Clearance Level enum)
  • failedAttempts: Integer
  • status: String (ACTIVE/LOCKED)
  • lastLogin: DateTime (UTC)
  • email/phone: String

Collection: security_logs

Immutable append-only ledger tracking every operational shift, access vector, and terminal command.

  • _id: ObjectId
  • eventType: String (e.g., FAILED_LOGIN)
  • username: String (Linked UID)
  • severity: String (HIGH/MEDIUM/LOW)
  • module: String (e.g., Auth/Access)
  • description: String (Event details)
  • sourceIP: String (IPv4 Trace)
  • timestamp: DateTime (Precision Logging)

Collection: alerts

Asynchronous threat notifications generated when system heuristics detect anomalies.

  • _id: ObjectId
  • alertId: String (e.g., TRK-9901)
  • severity: String (CRITICAL/HIGH)
  • title: String
  • assignedTo: String (Investigator UID)
  • createdAt: DateTime
  • status: String (OPEN/CLOSED)

Collection: risk_scores

Real-time quantification of threat levels assigned to specific nodes, systems, or operatives.

  • _id: ObjectId
  • category: String (Network/Identity)
  • score: Integer (Percentage 0 - 100)
  • entityId: String (Target Node/User)
  • factors: Array[String] (Vulnerabilities)
  • lastUpdated: DateTime

Collection: system_settings

Monolithic configuration documents managed via bulk 'upsert' operations altering global parameters instantly.

  • _id: ObjectId
  • key: String (auth_policy/ui/logging)
  • value: BsonDocument (Nested Payload)
  • description: String
  • updatedAt: DateTime
